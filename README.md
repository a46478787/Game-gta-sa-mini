# GTA San Andreas Mini - Русская версия

🚗 **Простая 2D версия игры в стиле GTA San Andreas для слабых компьютеров!**

## 📖 Описание

Это 2D игра с видом сверху в стиле GTA San Andreas, созданная специально для работы на минималках. Игра включает в себя:

- **Открытый мир** с городом, дорогами и зданиями
- **Реалистичную физику** машины с поворотами и инерцией
- **Систему миссий** с различными заданиями
- **NPC машины** и трафик
- **Систему розыска** с полицией
- **Смену дня и ночи**
- **Звуковые эффекты** и визуальные уведомления

## 🎮 Версии игры

### 1. **game.py** - Базовая версия
- Простая игра с основными функциями
- Подходит для самых слабых компьютеров
- Основные миссии и управление

### 2. **gta_enhanced.py** - Расширенная версия
- Полный функционал с NPC, полицией, системой розыска
- Смена дня и ночи, окна в зданиях
- Больше миссий и возможностей
- Читы и дополнительные функции

## 🚀 Установка и запуск

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Запуск игры

**Базовая версия:**
```bash
python game.py
```

**Расширенная версия:**
```bash
python gta_enhanced.py
```

## 🎯 Управление

| Клавиша | Действие |
|---------|----------|
| **W/A/S/D** или **Стрелки** | Движение машины |
| **R** | Рестарт игры |
| **ESC** | Выход из игры |
| **SPACE** | Увеличить розыск (для тестирования) |
| **M** | Добавить деньги (чит-код) |

## 🎪 Особенности игры

### 🚗 Физика машины
- Реалистичная инерция и торможение
- Поворот только при движении
- Разная скорость для игрока и NPC

### 🌆 Городская среда
- Процедурно генерируемые здания
- Дорожная сеть с разметкой
- Разные типы зданий (офисы, дома, магазины)

### 📋 Система миссий
- Различные типы заданий
- Награды за выполнение
- Указатели направления к цели

### 🚔 Система розыска
- Уровень розыска от 1 до 5 звёзд
- Появление полицейских машин
- Преследование игрока

### 🌅 Смена времени суток
- Динамическое изменение цвета неба
- Освещение зданий в ночное время
- Влияние на видимость

## 🎮 Геймплей

### Цель игры
Выполните все миссии, перемещаясь по городу и избегая полиции. Зарабатывайте деньги и очки за выполнение заданий.

### Миссии
1. **Доберись до банка** - Первое задание (+200$)
2. **Найди магазин** - Торговый центр (+150$)
3. **Посети больницу** - Медицинская помощь (+300$)
4. **Вернись домой** - Возвращение (+250$)
5. **Встреча в центре** - Финальная миссия (+500$)

### Советы по игре
- 🚗 Не превышайте скорость слишком часто, чтобы не привлечь полицию
- 🎯 Следите за стрелкой-указателем к цели миссии
- 🏃 Используйте телепортацию через края экрана для быстрого перемещения
- 💰 Выполняйте миссии последовательно для максимальной награды

## 🛠️ Системные требования

### Минимальные:
- **ОС:** Windows 7/8/10/11, Linux, macOS
- **Процессор:** Любой двухъядерный процессор
- **Память:** 2 GB RAM
- **Видеокарта:** Встроенная графика
- **Место на диске:** 50 MB

### Рекомендуемые:
- **Процессор:** Intel i3 или AMD equivalent
- **Память:** 4 GB RAM
- **Видеокарта:** Любая дискретная карта

## 🐛 Известные проблемы

- Иногда NPC машины могут застревать в углах
- При очень высоком FPS может быть слишком быстрое движение
- Русские шрифты могут отображаться некорректно на некоторых системах

Если у вас возникли проблемы с запуском игры:

3. Попробуйте базовую версию игры если расширенная тормозит
