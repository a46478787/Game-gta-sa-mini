# GTA San Andreas - Минимальная версия

Простая 3D игра в стиле GTA San Andreas, созданная с использованием Three.js.

## Особенности

- 🎮 3D мир с зданиями и машинами
- 👤 Персонаж с управлением от первого лица
- 🚗 Взаимодействие с объектами
- 🏃‍♂️ Движение и прыжки
- 🎯 Система здоровья
- 🌆 Простой городской ландшафт

## Управление

- **WASD** - Движение
- **Мышь** - Осмотр вокруг
- **Пробел** - Прыжок
- **E** - Взаимодействие с объектами
- **Клик** - Заблокировать курсор мыши

## Запуск

1. Откройте `index.html` в современном браузере
2. Кликните на экран для блокировки курсора
3. Используйте WASD для движения и мышь для осмотра

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Three.js для 3D графики
